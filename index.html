🗣️ Useful Sentence Patterns

Use these complete structures to explain your ideas!

🎨 1. Colours & Base

  • The main colour I chose is...
  • It is designed in a classic...
  • I decided to go with a ... base.

🌟 2. Features

  • One of the key features is...
  • To make it stand out, I included...
  • My favourite part of the bag is...

📍 3. Locations

  • It is placed in the centre.
  • You can find it located on the left.
  • I decided to put it...

🤔 4. Giving Reasons

  • I chose this design because it looks...
  • This design is special to me because...
  • The reason behind this design is that...
